## Project Description

Built a mail view with basic functions archiving and reading of mail

## Prerequisite
- nodejs
- vue 3.x.x

## Setup Instructions
- Run `npm install` to install all dependancies
- `npm run serve` to run dev environment
- `npm run build` to build project into`/dist` folder
